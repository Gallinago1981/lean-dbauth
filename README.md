# Spring Security DB認証

DBが保持しているユーザ情報を用いてのログインおよびログイン後の処理を行う。

## 環境
- JDK11
- Spring Boot 2.1.5
- Spring Security
- H2Database
- Mybatis

## 機能概要

- ログイン
  - DBが保持しているユーザID、パスワードでの認証を行う。
  - パスワードはbcryptで暗号化
- ログインユーザの情報による各種制御
  - パスワード再設定
    - パスワードの有効期限が切れていた場合はパスワード再設定画面を表示する
  - 無効アカウント
    - アカウントが無効化されている場合はエラー画面を表示する
